Learning Path

  1. Hangul — write & read consonants (초성/종성), vowels (중성), and build syllables.
  2. Common Words — numbers, family, food, school, time.
  3. Phrases — greetings, introductions, requests, likes/dislikes.
  4. Sentence Structure — SOV order, particles (은/는, 이/가, 을/를), locations/time, present tense.
